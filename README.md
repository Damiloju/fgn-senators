# fgn-senators

An application and api to get the details of senators of the federal repulblic of nigeria

# API endpoints

1. `GET` `/senators`

   - Get all senators

2. `GET` `/senators`

   - Get all senators

3. `GET` `/senator`

   - Get senator by name
     - parameters `Name`

4. `GET` `/senators/:state`
   - Get senators by state
